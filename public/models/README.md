# /public/models/

Place your GLTF/GLB 3D model file here as:

  tshirt.glb

## Getting a Free T-Shirt Model

Option 1 — Sketchfab (free):
  https://sketchfab.com/search?q=tshirt&type=models&features=downloadable&license=cc-attribution

Option 2 — Ready Player Me shirt (free):
  Download from https://github.com/mrdoob/three.js/tree/dev/examples/models

Option 3 — Use the built-in fallback:
  The app ships with a procedural box-geometry fallback that works out of the box
  with no model file needed. The fallback is active when tshirt.glb is missing.

## Model Requirements
- Format: GLTF 2.0 (.glb preferred)
- Scale: ~2-3 units tall
- Mesh name: "tshirt" (or any — first mesh is auto-detected)
- UV-mapped for texture projection
- Recommended poly count: 5k–30k for performance
